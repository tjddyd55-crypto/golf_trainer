# 매장 등록 요청 승인 페이지 URL

## Railway 배포 URL

슈퍼 관리자 서비스는 `golf-super-admin` 서비스로 배포됩니다.

### 기본 URL 확인 방법
1. Railway 대시보드 > `golf-super-admin` 서비스 선택
2. Settings 탭 > Domains 섹션에서 URL 확인
3. 또는 Deployments 탭에서 최신 배포의 URL 확인

---

## 매장 등록 요청 승인 페이지

### URL
```
https://[golf-super-admin-railway-url]/store-requests
```

### 기능
- 승인 대기 중인 매장 등록 요청 목록 표시
- 각 요청에 대해 **승인** 또는 **거부** 버튼 제공
- 승인 시 타석 자동 생성 및 매장 활성화
- 거부 시 매장 상태를 'rejected'로 변경

### 접근 방법
1. 슈퍼 관리자 로그인
2. 대시보드에서 **"매장 등록 요청"** 버튼 클릭
3. 또는 직접 `/store-requests` URL 접근

---

## 사용 흐름

1. **매장 등록 요청**
   - 매장 관리자가 `/signup`에서 매장 등록
   - 상태: `pending` (승인 대기)

2. **슈퍼 관리자 승인**
   - `/store-requests` 페이지에서 요청 확인
   - 승인 클릭 → 타석 생성 및 매장 활성화
   - 거부 클릭 → 매장 거부 처리

3. **매장 관리자 로그인**
   - 승인된 매장만 로그인 가능
   - `pending` 상태: "승인 대기 중" 메시지
   - `rejected` 상태: "거부됨" 메시지

---

## 대시보드 링크

슈퍼 관리자 대시보드에서 **"매장 등록 요청"** 버튼을 클릭하면 바로 이동합니다.

승인 대기 중인 요청이 있으면 빨간색 배지로 개수가 표시됩니다.
