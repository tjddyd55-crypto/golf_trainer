# 유저 서비스 URL 정보

## Railway 배포 URL

유저 서비스는 `golf-user` 서비스로 배포됩니다.

### 기본 URL 확인 방법
1. Railway 대시보드 > `golf-user` 서비스 선택
2. Settings 탭 > Domains 섹션에서 URL 확인
3. 또는 Deployments 탭에서 최신 배포의 URL 확인

---

## 주요 페이지 URL

### 1. 회원가입
```
https://[golf-user-railway-url]/signup
```

**입력 항목:**
- 이름
- 생년월일 (날짜 선택)
- 성별 (남/여)
- 연락처 (휴대폰번호, 하이픈 포함)
- 비밀번호

**중복 체크:** 휴대폰번호로 중복 가입 방지

---

### 2. 로그인
```
https://[golf-user-railway-url]/login
```

**입력 항목:**
- 아이디 (휴대폰번호, 하이픈 제외)
- 비밀번호

**로그인 후:** 매장/타석 선택 화면으로 자동 이동

---

### 3. 매장/타석 선택
```
https://[golf-user-railway-url]/select-store-bay
```

**기능:**
- 매장 선택 (드롭다운)
- 타석 선택 (드롭다운, 매장 선택 시 자동 로드)
- 확인 버튼 클릭 시 "매장/타석 이용중" 메시지 표시
- 활성 세션 등록 (매칭)

---

### 4. 메인 페이지
```
https://[golf-user-railway-url]/main
```

**표시 내용:**
- "매장/타석 이용중" 상태 표시
- 최근 샷 데이터 (총거리, 캐리, 스매시팩터, 볼스피드)
- 연습 날짜 목록
- 전체 샷 기록 보기 링크

---

### 5. 전체 샷 기록
```
https://[golf-user-railway-url]/shots
```

**표시 내용:**
- 모든 샷 데이터 목록
- 색상 코딩 (좋음/경고/나쁨)

---

## API 엔드포인트

### 타석 목록 조회
```
GET /api/get_bays?store_id=[매장ID]
```

### 활성 사용자 조회
```
GET /api/active_user?store_id=[매장ID]&bay_id=[타석ID]
```

---

## 사용 흐름

1. **회원가입** (`/signup`)
   - 이름, 생년월일, 성별, 연락처, 비밀번호 입력
   - 휴대폰번호 중복 체크
   - 가입 완료 → 로그인 페이지로 이동

2. **로그인** (`/login`)
   - 아이디(휴대폰번호), 비밀번호 입력
   - 로그인 성공 → 매장/타석 선택 페이지로 이동

3. **매장/타석 선택** (`/select-store-bay`)
   - 매장 선택
   - 타석 선택
   - 확인 클릭 → "매장/타석 이용중" 표시
   - 메인 페이지로 이동

4. **메인 페이지** (`/main`)
   - "매장/타석 이용중" 상태 확인
   - 샷 데이터 확인
   - 전체 기록 보기

---

## 참고사항

- **아이디:** 휴대폰번호 (하이픈 제외, 예: 01012345678)
- **중복 가입 방지:** 휴대폰번호로 체크
- **매칭:** 매장/타석 선택 시 활성 세션에 등록되어 샷 수집 프로그램과 매칭됨
