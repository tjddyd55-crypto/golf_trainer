# 프로덕션 운영 체크리스트

Railway 서버로 전환 후 프로덕션 환경에서 운영을 시작하기 전 확인사항입니다.

---

## ✅ 1단계: Railway 서버로 전환

### 서버 URL 설정
- [ ] Railway 서버 URL 확인
- [ ] `main.py`의 서버 URL 업데이트
- [ ] `start_client.bat`의 서버 URL 업데이트

**실행 방법:**
```bash
python switch_to_railway.py
```

또는 수동으로:
1. `main.py`에서 `DEFAULT_SERVER_URL` 수정
2. `start_client.bat`에서 `SERVER_URL` 수정
